Wasta-Menus

Wasta-Menus: Main Menu Limiting Feature for Wasta-Linux

Wasta-Menus is a feature available to all Wasta-Linux users.  Essentially Wasta-Menus is a way to only display a limited number of applications in a computer's Main Menu.  Thanks go first and foremost to the Balsa team, whose implementation of a limited Application Menu in Balsa served as the basis for Wasta-Menus.  With Wasta-Menus enabled, a wasta [Linux] computer could serve as a kiosk computer or it could be used by beginning users who only have only been assigned a few specific tasks on the computer.  In both cases it is preferable to not overwhelm the user with ALL the installed wasta [Linux] applications in the Main Menu, but rather present a limited set of them to the user.

The following link is a very basic screen sharing video tutorial on how to use Wasta-Menus (approximately 7 minutes): https://www.youtube.com/watch?v=nzYMh3U9dZY

update 2014-07-15: Wasta-Menus can now be launched without a terminal (an administrative password is required), so the above video is a bit outdated.  See notes below on using Wasta-Menus

Instructions on using Wasta-Menus:

There are 2 applications needed to use Wasta-Menus, both described below.  Both applications require an administrative password to keep curious users from being able to make modifications.

  • Wasta-Menus Config: use this application (found in the Main Menu) to enable, disable, or fully remove the current user's Wasta-Menus.  Here is a screenshot:Simply click "On" and then click "OK" to enable a user's Wasta-Menus.  Then click on the Main Menu and you will see that only a limited number of applications are displayed.  The default list of applications in Wasta-Menus can be adjusted by a regional customizer of wasta [Linux].

  • Wasta-Menus Add Application: use this application (again, it is found in the Main Menu) to add an additional application to the current user's Wasta-Menus so that it will show in the Main Menu.  Here is a screenshot:
    Simply select the applications you would like to appear in the Main Menu and click "OK".  The applications should now appear in the Main Menu.

  • How to hide an application from the Main Menu: If you would like to hide an application from the Main Menu, this is easily done by using the standard Linux Mint Menu Editor.  Open this by right clicking on the Main Menu, selecting "Configure" and then clicking "Open the menu editor".  Here is a screenshot:

    You can uncheck a category on the right to hide the entire category from the Main Menu, or you can select a specific category on the left and then uncheck an individual application in the list on the right to hide an individual application from the Main Menu.
