Wasta-Linux: Downloads

Please reference the following when deciding what version of Wasta-Linux may be best for you.

Wasta-Linux SIL Software Compatibility Summary:

Both LTS (long-term support) versions (14.04 and 12.04): Most SIL Software that supports Linux is running equally well in both LTS versions of Wasta-Linux.  This includes Paratext 7.5, Fieldworks, Adapt It, ibus-kmfl ("Keyman"), Bloom, and WeSay.  TW5 (the "Folio" version only) and BART can be installed via "wine" and run acceptably.

Wasta-Linux Downloads:

  • 64bit is recommended for nearly all newer hardware, 32bit is required for some older hardware

Wasta-Linux 14.04.3

  • Notes:
    • Based on Ubuntu 14.04 "Trusty" / Linux Mint 17.2 "Rafaela"
    • Default Interface: Cinnamon
    • Includes the updated 3.19 Linux Kernel and graphics support (the same that Ubuntu 14.04.3 uses)
    • Security updates provided through April 2019
  • Wasta-Linux-14.04.3-64bit (updated 2015-08-24)
    • md5: c3ae22f0c6c62b4175b9c1d60fb58222
  • Wasta-Linux-14.04.3-32bit (updated 2015-08-24)
    • md5: 24310189b5a0cb0eacb6b799deb33b0b

Wasta-Linux 16.04.0

  • Notes:
    • Based on Ubuntu 16.04 "Xenial"
    • Interface: Cinnamon is the default interface, with Unity also installed
    • Security updates only provided through April 2021

Wasta-Linux 12.04.5

  • Notes:
    • Based on Ubuntu 12.04.5 "Precise" / Linux Mint 13 "Maya"
    • Default Interface: Cinnamon
    • Includes the updated 3.13 Linux Kernel and graphics support (the same that Ubuntu 12.04.5 uses)
    • Use on most hardware.  Some older machines may not be able to boot this newer kernel, so please use the Wasta-Linux 12.04.0 if this is the case
    • Security updates provided through April 2017
  • Wasta-Linux-12.04.5-64bit (updated 2014-07-22)
    • md5: c37032c9a2c10327fd796d87247c9f84
  • Wasta-Linux-12.04.5-32bit (updated 2014-07-23)
    • md5: 106ee654a04c82133987f111738e88c7

Wasta-Linux 12.04.0

  • Notes:
    • Based on Ubuntu 12.04 "Precise" / Linux Mint 13 "Maya"
    • Default Interface: Cinnamon
    • Includes the original 3.2 Linux Kernel and graphics support
    • Use only in the case that Wasta-Linux 12.04.5 does not boot due to older hardware limitations.
    • Security updates provided through April 2017
  • Wasta-Linux-12.04.0-64bit (updated 2014-07-25)
    • md5: 72373d31ab5af77f92b7db8c681adea2
  • Wasta-Linux-12.04.0-32bit (updated 2014-07-23)
    • md5: e28e9176ef5cbdf977f734d0347a9ab3
These images can be put on a USB flash drive or DVD just like a standard Ubuntu or Linux Mint ISO.
