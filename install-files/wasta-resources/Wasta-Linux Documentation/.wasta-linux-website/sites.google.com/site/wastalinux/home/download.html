Wasta-Linux: Downloads

Several have inquired recently about recommendations for which Wasta-Linux version they should use.  The questions are mainly about if Wasta-Linux 14.04 is stable enough to use or if they should stay with 12.04?  Unfortunately, due to some issues with some of the SIL Linux software applications running well in Wasta-Linux 14.04 (specifically with the default "Cinnamon" desktop), I have some recommendations, but it depends on your needs.

Please reference the following when deciding what version of Wasta-Linux may be best for you.

Wasta-Linux SIL Software Compatibility Summary:

Both versions: Most SIL Software that supports Linux is running equally well in both versions of Wasta-Linux.  This includes Paratext 7.4/7.5, Fieldworks (see below for one issue with Wasta-Linux 14.04 and Fieldworks), Adapt It, ibus-kmfl ("Keyman"), and WeSay.  TW5 (the "Folio" version only0) and BART can be installed via "wine" and run acceptably.

Wasta-Linux 14.04 concerns: Wasta-Linux 14.04 is working well and is considered "stable", and will be supported with security updates through April 2019.  However, there are few problems with some SIL software applications running in Wasta-Linux 14.04:

  • Bloom: a critical bug that affects Cinnamon only means that Bloom is virtually unusable in Wasta-Linux 14.04.  See this bug report for more details.  Other desktop choices like Unity or Gnome-Shell do not have this problem, the problem is only with the Cinnamon desktop of Wasta-Linux, and is only in version 14.04.  So, in Wasta-Linux 12.04, Bloom is running equally well in Cinnamon as with other desktop choices (Unity, Gnome-Shell, etc.)

  • Fieldworks: "auto-switching of keyboards based in input field" will not work currently in Wasta-Linux 14.04.  You can use the system tray icon or the "ibus-kmfl keyboard shortcut" to toggle keyboards independently, and then ibus-kmfl ("Keyman") keyboarding will work fine in Fieldworks, but the Fieldworks programmatic switching of keyboards will not work.  In Wasta-Linux 12.04, Fieldworks is running equally well with other desktop choices (Unity, Gnome-Shell, etc.)  Additionally, automatic keyboard switching in WeSay is working fine in Wasta-Linux 14.04.

Wasta-Linux 12.04 concerns: In Wasta-Linux 12.04, the only concern is that the there are some possible future bugs that may not be as easily solved as in 14.04 due to the older libraries and dependencies of the Ubuntu 12.04 base.  As of right now I am not aware of any specific bugs or concerns with SIL Software in Wasta-Linux 12.04 that is a concern.  Wasta-Linux 12.04 will be supported with security updates through April 2017.

Wasta-Linux Version Recommendations:

Use Wasta-Linux 14.04 if:

  • You will not use Wasta-Linux for language work.

  • You will use Wasta-Linux for language work, but use of Bloom or "automatic keyboard switching in FLEx" are not necessary.

  • You will use Wasta-Linux for langauge work, including Bloom and FLEx, but you will opt for an alternative desktop other than the default "Cinnamon" desktop of Wasta-Linux 14.04.  At this point there are no "wasta customizations" for any other desktop, so you would need to work through any usability issues found using a different desktop independently.

  • You have *brand new hardware* (designed in 2014 or newer such as the new HP Stream laptops) which requires the "Linux 3.16 kernel".  The Linux 3.13 kernel, which was the default Linux kernel with Ubuntu 14.04 is already in Wasta-Linux 12.04 so Wasta-Linux 14.04 will support most new hardware.  Yes, you are reading correctly that Wasta-Linux 12.04 is using the Ubuntu 14.04 default kernel .  Even most brand new laptops will have all of their hardware operate just fine under the Linux 3.13 kernel, so this is only for a specific laptop that is found to have hardware issues with the older kernel.

  • You are OK applying any possible future updates to Wasta-Linux 14.04 that may solve any SIL software application compatibility issues.

Use Wasta-Linux 12.04 if:

  • You will use Wasta-Linux for language work and will use Bloom and/or want to ensure "automatic keyboard switching in FLEx" is working well.

  • You will deploy some Wasta-Linux machines for language work and will use Bloom and/or want to ensure "automatic keyboard switching in FLEx" is working well and you want to have non-language work Wasta-Linux machines be identical in setup.

  • You want the most stable (meaning "unchanging") Wasta-Linux experience.  At this point, only minor usability changes may possibly be made to Wasta-Linux 12.04.

  • The Linux 3.13 kernel (or older: Wasta-Linux 12.04 can be downgraded to any of the 3.2, 3.5, 3.8, or 3.11 Linux kernels) is adequate for your installations.


With this information, I hope you can make the best decision regarding what version of Wasta-Linux to use for your location.
