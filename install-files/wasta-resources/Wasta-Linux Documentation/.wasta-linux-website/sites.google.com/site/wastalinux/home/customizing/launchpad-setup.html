Launchpad Setup

Wasta-Linux Launchpad Setup

This page is part of the Wasta-Linux customizing process.

Launchpad is Ubuntu's site for hosting code.  The Wasta-Linux Launchpad PPA is where the custom packages for Wasta-Linux live.

Follow the below steps in order establish your Launchpad ID and sign the Ubuntu Code of Conduct, which is required before you are able to upload any packages to a Launchpad PPA.

After the Code of Conduct is signed, your Launchpad ID can be added as a Wasta-Linux "team member", allowing you to maintain your regional wasta-custom-* packages.

1. Sign up for a Launchpad ID: https://launchpad.net/

2. Create an OpenPGP key and import it to Launchpad:

  • "Passwords and Keys" app: "File | New" to create an OpenPGP Key
    • IMPORTANT: do NOT enter a "key comment", only enter "name" and "email", or else there will be problems with using it later.
    • Setting the key to never expire should be OK.
  • "Passwords and Keys" app: "Remote | Sync and Publish Keys" in order to sync your OpenPGP Key to the Ubuntu Keyserver.
  • Your Launchpad account page: Add your OpenPGP key (see red box below, click yellow edit circle), then follow instructions:
  • Part of the process in uploading the OpenPGP key will be receiving and decrypting a secure email.  You will need the "enigmail" extension for Thunderbird to do this the easy way.
  • Reference Document for more information: https://launchpad.net/+help-registry/openpgp-keys.html#publish

3. Sign Ubuntu Code of Conduct:

  • Your Launchpad account page: "sign" Ubuntu code of Conduct.  See red box below, click "No", follow instructions:

4. Contact the Wasta-Linux team

Please contact us through the Wasta-Linux Users Forum after you have signed the Ubuntu Code of Conduct so we can add your Launchpad ID as a Wasta-Linux Launchpad team member.
